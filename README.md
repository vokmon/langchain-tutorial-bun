# Langchain with bun

Init with bun
```bash
bun init
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

```bash
bun run src/run/main.ts
```

Use this for init eslint

```
npm init @eslint/config@latest
```


View bun lock file

```
*.lockb binary diff=lockb
```
